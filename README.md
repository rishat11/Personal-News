# Personal News Digest (Telegram bot, MVP)

## Setup

1. Install Python 3.12+ (project is written for 3.12; works locally with newer versions too).
2. Create `.env` based on `.env.example`.

## Install deps

```bash
python -m pip install -r requirements.txt -r requirements-dev.txt
```

## Run

```bash
python -m src.main
```

## Tests

```bash
python -m pytest -q
```
